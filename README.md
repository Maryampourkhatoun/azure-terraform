# Azure Infrastructure mit Terraform

## Beschreibung
Automatische Bereitstellung einer Azure
Resource Group und eines Storage Accounts
mit Terraform und einheitlichem Tagging-Schema.

## Technologien
- Terraform
- Microsoft Azure
- Infrastructure as Code (IaC)

## Erstellte Ressourcen
- Azure Resource Group
- Azure Storage Account (Standard LRS)

## Verwendung
terraform init
terraform plan
terraform apply

## Aufräumen
terraform destroy
